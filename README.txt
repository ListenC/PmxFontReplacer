PmxFontReplacer 插件（目标：将 "MS UI Gothic" 和 "Yu Gothic UI" 替换为 "宋体"）
============================================================================================

说明：
- 这个插件在运行时遍历已加载的程序集，尝试替换静态 Font 字段/属性中使用的字体名为目标字体。
- 同时递归遍历打开的窗体与控件，替换控件的 Font。对 ToolStrip 项也会替换。
- 会监听程序集加载（AssemblyLoad）处理动态创建的控件或后续加载的程序集。

2025.9.15更新：
- 递归替换多级菜单字体，解决个别深层路径字体无法替换问题。
- 编译Relase版本减少插件大小。

使用方法：
将解压得到的 DLL 放到 PMXEditor 的插件目录（Plugins）下，启动 PMXEditor。

注意事项：
- 这是运行时替换（不会修改原始 DLL 文件）。若需要修改 DLL 内嵌资源或对内部 new Font 调用拦截，需要更高级的 IL patch/注入。
- 如果系统中没有 宋体 ，替换会回退到系统默认字体。
- 运行时会在DLL路径下生成PmxFontReplacer.config配置文件。
